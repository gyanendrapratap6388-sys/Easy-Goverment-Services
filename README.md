# EasyGov Services Website

EasyGov Services is a simple responsive website built using HTML, CSS and JavaScript.  
This website helps users easily access online government related services like job form filling, MSME registration, GST registration, and certificate applications.

## 🚀 Features
- Responsive modern website design
- Government service information
- Contact form for service requests
- WhatsApp direct contact button
- Service cards for different services

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts

## 📂 Project Structure
- index.html → Main website file
- CSS (inside style tag) → Website design and layout
- JavaScript → Contact form submission message

## 💼 Services Included
- Online Job Form Filling
- MSME Registration
- GST Registration
- Aadhar Update
- Income Certificate
- Caste Certificate
